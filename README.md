# share-modale
A minimal profile share page

# Pre-requisites

this project uses,

- [vitejs](https://vitejs.dev/)

- [nodejs](https://nodejs.org/en)

- [TailwindCSS](https://tailwindcss.com/)


# Running Locally

To run this project on live server, run

`npm run start`

# Build for deploy

To build this file, run

`npm run build`
